# Foreign Library Interface legacy vignette

This directory preserves the original `FLI.Rnw` source, bibliography and image
assets for the historical "Foreign Library Interface" article by Daniel Adler.

The material is no longer built as the package vignette. It describes the older
rdyncall/DynPort design and remains useful as historical background and source
material for future documentation, but it should not be treated as the current
user-facing tutorial for the package.
